    Команды PON

    При вводе команд на OLT можно использовать авто дополнение при помощи клавиши Tab Можно посмотреть какие доступны параметры для команды при помощи клавиши ? Команды категории show позволяют отобразить информацию Для входа в режим конфигурации используется команда config Для удаления строки из конфигурации используется аналогичная строка с параметром no в начале

    Просмотр информации и статуса ONU:

    В качестве параметров можно передать MAC адрес ONU, разделённый точками

    show epon onu-i mac abcd.ef01.2345

    Просмотр всех ONU на определённом интерфейсе:

    show epon onu-i in epon0/1

    Просмотр всех активных ONU на определённом интерфейсе:

    show epon act in epon0/1

    Просмотр всех неактивных ONU на определённом интерфейсе:

    show epon ina in epon0/1

    Краткая информация о всех ONU на OLT:

    sh epon onu-status-count

    Просмотр конфигурации:

    Параметр non-? позволяет скрыть часть конфигурации, к примеру epon интерфейсы или llid ONU

    sh run

    Просмотр конфигурации конкретного интерфейса:

    sh run int epon0/1

    Просмотр конфигурации конкретной ONU:

    sh run int epon0/1:1

    Команды для ONU: Просмотр таблицы MAC адресов активной ONU:

    show mac ad dy in epon0/1:1

    Просмотр мощности приходящего оптического сигнала на ONU:

    show epon in epon0/1:1 onu ctc opt

    Просмотр мощности исходящего оптического сигнала от ONU:

    show epon opt int epon0/1:1

    Просмотр мощности приходящего КТВ сигнала на ONU:

    show epon int epon0/1:1 onu catv-in

    Просмотр состояния ethernet порта на ONU:

    show epon int epon0/1:1 onu port 1 state

    Редактирование конфигурации:

    Для редактирования конфигурации необходимо предварительно зайти в режим конфигурирования, в зависимости от задачи также может понадобится зайти в конфигурацию конкретного интерфейса. Включение\отключение ethernet порта на ONU:

    conf
int epon0/1:1
epon onu port 1 ctc shutdown
no epon onu port 1 ctc shutdown

    Удаление\привязка ONU на epon, в параметре sequence указывается порядковый номер ONU, можно заменить параметр sequence на mac и указать MAC ONU которую необходимо удалить:

    conf
int epon0/1
no epon bind-onu sequence 1
epon bind-onu mac

    Полисер скорости: Параметр pir - peak bandwith (указывается в соответствии с тарифом) Параметр cir - commited bandwith, указываем 10000 При тарифе 100Мб/c pir = 100000 При тарифе 300 pir = 350000 При тарифе 500 pir = 550000 При тарифе 800 pir = 850000 При тарифе 1000 pir = 1000000

    conf
int epon0/1:1
epon sla upstream pir 550000 cir 10000
epon sla downstream pir 550000 cir 10000

    Блокировка\разблокировка регистрации ONU:

    conf
int epon0/1
epon onu-blacklist mac
no epon onu-blacklist mac

    Удаление всех ONU со статусом lost:

    clear epon lost-onu

    Привязка шаблона:

    conf
int epon0/1
epon pre-conf Giga300 binded-onu-llid 1 param 2125

    Для корректной привязки шаблона, необходимо удалить ONU, чтобы она зарегистрировалась с новым шаблоном, в binded-onu-llid указывается номер ONU, в param указывается клиентский vlan на этом epon Работа с логами: Просмотр логов:

    sh log

    Просмотр логов с определённого времени до текущего момента: hh - часы mm - минуты yyyy - год MM - месяц dd - день

    sh log start hh mm yyyy MM dd

    Поиск в строках осуществляется при помощи спец. символа |, многие команды его поддерживают, после написания команды можно в конце указать | и задать параметры для поиска: begin - будут выведены строки начинающиеся с указанного параметра include - будут выведены строки содержащие указанный параметр exclude - будут скрыты строки содержащие указанный параметр

    sh log | include EPON0/1:1
